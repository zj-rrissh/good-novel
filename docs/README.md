# 模块文档索引

本目录存放智能小说平台的模块级详细设计文档，后续开发以本目录文档为准。

统一文档约束：

1. 每个模块文档必须包含：模块目标、职责边界、模块结构、核心流程、接口约束、数据约束、缓存/事务/安全约束、异常与日志约束、实现边界、开发顺序。
2. 文档只描述设计与约束，不包含具体代码实现。
3. 模块之间通过接口和领域边界协作，禁止跨模块直接操作对方内部实现。
4. 若总方案与模块文档冲突，以模块文档经汇总修订后的版本为准，并同步回写总方案。
